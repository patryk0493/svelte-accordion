## 📔 Installation

`npm install -D svelte-epic-accordion`

`pnpm i -D svelte-epic-accordion`

`yarn add -D svelte-epic-accordion`

## ❓ How to use❓

```svelte
<script>
  import Accordion from "svelte-epic-accordion";
</script>

<Accordion let:Section>
	<Section id='one'>
		Section one
	</Section>
	<Section id='two'>
		Section two
	</Section>
</Accordion>
```
